# Workout Tracker
A web-based workout tracking application built with FastAPI and SQLAlchemy.

Users can create workouts, log exercises, and track history

---

### Tech Stack

Backend
 - Python
 - FastAPI
 - SQLAlchemy
 - Pydantic

Database
 - SQLite (Development)
 - PostgreSQL (Planned)

Tools
 - Git
 - pytest

---

### Current Features

 - Basic REST API endpoints
 - SQLAlchemy relational db models
 - User authentication (OAuth2)
 - Logging workouts and sets

---

### Planned Features

 - Workout templates
 - Prebuilt/Custom exercises
 - - User selects proper exercise
 - - Exercise contains information like: equipment, muscle groups
 - Workout history
 - Email based password recovery
 - Frontend GUI
 - Analytics
